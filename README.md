episodes.html：背景渐变色的简单设计，jQuery翻转卡片式插件的使用

classic.html：本地json文件的读取，点击标语跳转对应的出处页面

extra.html：东方主题，一些图片的特效，邮件发送功能，两处localStorage的实现
